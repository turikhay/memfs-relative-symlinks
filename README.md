memfs vs Node.js
===================

> [!NOTE]  
> This issue was fixed in 4.15.1

See [this issue](https://github.com/streamich/memfs/issues/725) for the context.

Run to see an issue with memfs:
```bash
$ node memfs.js
```

Run to create real symlinks with Node.js:
```bash
$ node nodefs.js
$ tree test
```
